结合你这套 **Elysia + Vue3 + Naive UI** 技术栈，我给你整理一份**最实用、企业级后台必备、做模板一次写完终身复用**的功能清单。
不搞花里胡哨，全部是**真实项目 100% 会用到**的核心功能。

# 一、登录与权限（必做）

1. **账号密码登录**
2. **验证码 / 图形验证码**
3. **记住我、自动登录**
4. **JWT 鉴权 + Token 刷新机制**
5. **角色管理（RBAC）**
    - 超级管理员
    - 运营/编辑/普通管理员
6. **菜单权限**（不同角色看到不同菜单）
7. **按钮权限**（新增/删除/编辑/导出控制）
8. **操作日志**（谁在什么时候干了什么）

# 二、系统基础设置（必做）

1. **用户管理**
    - 列表、搜索、新增、编辑、禁用、重置密码
2. **角色管理**
    - 权限勾选面板
3. **菜单管理**
    - 动态路由、支持多级菜单、图标配置
4. **字典/配置管理**
    - 全局常量（分类、状态、开关）
5. **文件上传 / 图片管理**
    - 单图、多图、富文本图片
6. **系统设置**
    - 网站名称、LOGO、版权、开关配置

# 三、业务数据管理（核心）

1. **列表页标配**
    - 筛选
    - 分页
    - 排序
    - 批量操作
    - 导出 Excel
2. **表单页标配**
    - 新增
    - 编辑
    - 校验
    - 富文本
    - 图片上传
3. **详情页**
4. **状态切换**（启用/禁用）
5. **排序拖拽**
6. **回收站 / 软删除**

# 四、常用业务模块（通用必做）

1. **内容管理（文章/公告）**
2. **评论管理**
3. **Banner / 首页推荐**
4. **用户反馈 / 消息**
5. **订单/支付（如果涉及交易）**
6. **分类管理（无限级分类）**

# 五、UI 与体验组件（Naive UI 强项）

1. **布局**
    - 侧边栏 + 顶部导航 + 多标签页
2. **暗黑模式**
3. **全屏**
4. **菜单折叠**
5. **切主题色**
6. **面包屑**
7. **返回顶部**

# 六、后台必备的“小工具”

1. **401/404/500 错误页**
2. **接口请求封装**
3. **loading、异常、空状态统一处理**
4. **路由守卫**
5. **自动登出**
6. **复制文本、代码高亮**

# 七、你做模板最应该优先实现的（精简版）

如果你要做**通用模板**，不用做业务，只做骨架，实现这些就够 90% 项目直接用：

1. 登录 + 验证码
2. JWT 鉴权
3. 路由守卫
4. 用户管理
5. 角色管理
6. 菜单管理（动态路由）
7. 按钮权限指令
8. 系统配置
9. 文件上传
10. 列表 + 表单 + 分页 + 导出模板
11. 布局（侧边栏 + tabs + 暗黑模式）

做到这些，**任何后台项目你都能 10 分钟搭完架子**。
