# 1.01 已完成功能清单

> 本文档用于对外展示 Harbor 当前“已可用”的基座能力。

## A. 认证与权限

- JWT 登录/刷新/登出链路已打通。
- RBAC 管理闭环已具备：用户、角色、菜单、权限、部门。
- 管理端权限控制已落地：路由保护 + `v-permission` 指令。
- 系统保留角色保护已落地：避免种子角色被误删/误禁用。

## B. 系统稳定与治理

- 统一错误码体系 + 前端错误映射机制已落地。
- `/ready` 就绪检查可返回数据库/Redis 状态。
- 分级限流与 CORS 白名单策略已接入。
- 审计日志支持记录与多维筛选查询。

## C. 后台工程化能力

- CRUD 组件基建已沉淀：
  - `SearchBar`
  - `TableToolbar`
  - `FormDrawer`
  - `UnifiedState`
  - `DataTablePage`
- 用户管理、角色管理等核心页面已模板化。
- 用户管理支持动态角色下拉（按启用角色实时加载）。
- 字典管理后台化已落地：字典类型/字典项支持 CRUD 与状态切换。
- 部门管理最小闭环已落地：部门树查询、增删改、状态切换、子部门删除保护。
- 用户管理支持按部门筛选，并支持在新增/编辑时维护所属部门。

## D. 质量与交付体系

- 后端单元测试与集成测试可运行。
- Admin E2E smoke 已接入并可执行。
- CI 门禁覆盖 typecheck / lint / format / test / build。
- OpenAPI 自动生成与文档入口可用：`/openapi.json`、`/docs`。

## E. 文档与协作

- README（精简上手）已完成。
- 执行与治理文档持续维护（路线、台账、工作流）。
- 接口新增工作流、错误码字典等规范文档已可用。
